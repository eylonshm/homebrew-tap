# Homebrew Tap

Custom Homebrew formulae and casks.

## Installation

```bash
brew tap eylonshm/tap
```

## Available Casks

### claude-usage-widget

macOS menu bar app and desktop widgets for monitoring Claude Code usage and quota.

```bash
brew install --cask claude-usage-widget
```

Requires macOS 14 (Sonoma) or later and [Claude Code CLI](https://claude.ai/code).
